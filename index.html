Horus IA - Control de Asistencia

Como Project Manager, coordiné el desarrollo de HORUS, una aplicación móvil con IA para optimizar la asistencia estudiantil. El sistema gestiona listados y genera reportes automáticos, analizando patrones de asistencia para facilitar la toma de decisiones académicas.

Problemas Solucionados:
  • Registro manual
  • Rapidez en grupos grandes
  • Acceso histórico inmediato
